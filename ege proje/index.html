Notes
-Idle animation plays automatically when an animation is completed, others can be played with their assigned buttons.
-Used PIXI to render background and THREE for 3d objects on the scene.
-Used orbit controls for camera.Can be controlled via mouse buttons and mouse itself.left button to rotate, right button to move, scroll to zoom in and out
-I've used "Live Server" extension from Visual Code to test the project
Controls
W: Forwards Movement
S: Backwards Movement
A: Left Rotation
D: Right Rotation
Space: Jump
